TO-DO LIST

Below is an overview of the application architecture and functionality. Please familiarize yourself with it to get started

Requirements:

  • Create / Edit / Delete Completed tasks
  • Fields: Title, Category (Health, Home etc), Description, Due date, Complete / Incomplete flag, Search by title or category
  • Clicking on a ToDo task opens a detailed preview of the task without navigating away from the list
  • I’ll let you decide how you want to allow the user to ‘complete’ a ToDo (either from the list or from the preview window)
  • Bonus: List ToDo items with pagination (5 ToDo’s per page) – only show title and category when displaying the tasks in a list

Technology Stack

  • Spring boot 2.1 (Java 1.8)
  • Angular 7.2.2
  • CSS
  • H2 for development and MySQL for production
  • Github
